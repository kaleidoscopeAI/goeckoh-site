Privacy

Goeckoh is built around a simple principle: speech support should not require surveillance.

Summary

  • Offline‑first: core functionality is designed to run locally.
  • No ad tracking: we do not sell user data.
  • Minimal collection: we collect only what’s necessary to run the website and process purchases.

Website analytics

When you visit goeckoh.com, standard web server logs may record basic technical information such as IP address, browser type and request time for security and reliability. We do not use third‑party analytics services.

Payments

If you purchase through Stripe, Stripe processes payment information directly. We do not receive your full card number.

Children’s privacy

This product is built for families. If you are a parent or guardian and believe we have collected information improperly, contact support and we will take appropriate action.

Contact

For privacy questions: support@goeckoh.com (replace with your real support email).
