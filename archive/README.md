# Archive Directory

This directory contains archived materials for reference purposes. These files are not actively used in the current system but are preserved for historical reference and research.

## Directory Structure

### `/archive/research/`
Research papers, chat logs, technical discussions, and reference materials:
- PDF research papers on AGI, voice systems, and neurobiology
- Chat conversation logs with AI systems (Grok, Gemini, ChatGPT)
- Technical blueprints and architectural documents
- Voice system research and mathematical frameworks
- Autism research and intervention systems

### `/archive/old-docs/`
Older versions of documentation and configuration files:
- Previous versions of HTML/CSS files
- Deprecated package.json configurations
- Old requirements.txt files
- Previous system layouts

### `/archive/media/`
Archive media files:
- Audio recordings (MP3 files)
- Demonstration files
- Research audio samples

## Notes

- Files in this directory are NOT part of the active codebase
- Do not reference these files in production code
- Files are kept for historical reference and research purposes
- Some files may contain experimental or outdated information

## Usage

These archives may be useful for:
- Understanding the evolution of the system
- Reference for research discussions
- Historical context for design decisions
- Training materials and examples

To reference active documentation, see the main `/docs/` directory instead.
